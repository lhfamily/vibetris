# VIBETRIS

An educational Tetris game built with kids to learn web development fundamentals.

## About

VIBETRIS is a fully-featured Tetris clone with a retro aesthetic, complete with:
- **Retro CRT effects** - Scanlines and color separation for that authentic arcade feel
- **Dynamic music** - Korobeiniki (Tetris theme) synthesized in-browser using Web Audio API
- **Progressive difficulty** - Speed and music tempo increase with each level
- **Particle effects** - Visual feedback for line clears and game over
- **Mobile support** - Touch controls and responsive layout
- **Local highscore** - Track your best score using localStorage

## How to Play

### Desktop
- **Arrow Keys**: Move left/right, rotate (up), drop (down)
- Open `index.html` in your browser

### Mobile
- Touch the on-screen buttons to control pieces
- Tap or swipe on the game board for additional input

## Educational Goals

This project teaches:
- HTML5 Canvas drawing and animation
- JavaScript game loops and state management
- CSS animations and responsive design
- Web Audio API for synthesized music
- Touch event handling
- LocalStorage for data persistence

## Features

- Progressive level system (10 lines per level)
- Real-time score tracking
- Next piece preview
- Ghost piece indicator
- Splash screen with settings
- Game over screen with stats
- Mobile-optimized controls

## Running the Game

Simply open `index.html` in any modern web browser. No build tools or dependencies required.
